## Funcionalidades
- Lê um ficheiro de vocabulário
- Divide texto em palavras
- Calcula distância entre palavras
- Sugere correções para palavras erradas
- Mostra erros ortográficos num texto

## Ficheiros
- main.py → funções principais
- tests.py → testes do programa
- vocabulario.txt → lista de palavras

## Como executar
1. Instalar Python
2. Colocar todos os ficheiros na mesma pasta
3. Correr no terminal:
```
python testes.py
```
